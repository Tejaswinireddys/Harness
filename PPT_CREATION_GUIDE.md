# PowerPoint Creation Guide

## How to Create PPT from the Presentation Outline

### Option 1: Manual Creation
1. Open PowerPoint
2. Use the `CD_FLOW_PRESENTATION.md` file as your guide
3. Create slides following the outline (30 slides total)
4. Copy content from each slide section
5. Add diagrams and visuals as described

### Option 2: Using Online Tools
1. Convert Markdown to PPT using:
   - **Marp** (https://marp.app/) - Markdown to PPT converter
   - **Pandoc** - Command-line tool
   - **Slidebean** - Online presentation tool

### Option 3: Using PowerPoint Designer
1. Import the presentation outline
2. Use PowerPoint's Designer feature
3. Apply Harness-branded templates (if available)

## Color Scheme Suggestions

- **Primary Color:** #0066CC (Professional Blue)
- **Secondary Color:** #00A86B (Green for success)
- **Accent Color:** #FF6600 (Orange for highlights)
- **Background:** White or Light Gray
- **Text:** Dark Gray (#333333) or Black

## Font Recommendations

- **Title Font:** Arial Bold, 44pt
- **Subtitle Font:** Arial, 32pt
- **Body Font:** Calibri, 18-24pt
- **Code Font:** Courier New, 14pt

---

## Tips for Creating the PPT

1. **Keep it Visual:** Add diagrams, flowcharts, and icons
2. **Consistent Formatting:** Use same fonts, colors, and styles
3. **Limit Text:** Use bullet points, not paragraphs
4. **Use Animations Sparingly:** Only for emphasis
5. **Include Speaker Notes:** Add notes for each slide
6. **Test on Projector:** Ensure readability from distance
